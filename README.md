# Writing Testable Go

A hands-on workshop: from Go basics to professional testing patterns.

## Prerequisites

- Git
- Go 1.21+ (`go version` to check)
- Docker (optional, for integration tests)
- VS Code or any editor

## Quick Setup

```bash
git clone <repo-url>
cd writing-testable-go
go mod download
```

## Cloud Environment (No Install Required)

[![Open in Gitpod](https://gitpod.io/button/open-in-gitpod.svg)](https://gitpod.io/#https://github.com/workshop/writing-testable-go)

**Falling behind?** Checkout the `-complete` version of this stage/lab:
```bash
git checkout stage-1-complete
```

## Verify Setup

```bash
go version
# Should print: go version go1.21+ ...
```
